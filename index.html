Introduccion

Python es un lenguaje de programación que regularme despierte la sensación de ser muy "natural" o "simple", esto debe a su principal característica, que es la sintaxis sencilla, la facilidad de las reglas con las que cuenta el lenguaje lo hacen muy comprensible al momento de aprenderlo.

Python es un lenguaje completo que solo basta con entender su forma trabajar para que el usuario decida cómo implementar el lenguaje ya que en casi cualquier área del desarrollo de software se puede implementar.

Python como lenguaje es moderno e innovador, pero lo sorprendente es que este lenguaje está en el rango de edad de lenguajes como Java, PHP y otros a los que se les ve antiguos y es ahí lo sorprendente de Python, que sin importar la fecha en la que se aprenda nos asombran su diseño y sus capacidades de uso.

Durante mucho tiempo Python fue visto como un lenguaje de programación para academias, las matemáticas y la investigación. Con la aparición de "startups" en donde se necesita un tipo de tecnología distinta para el desarrollo de software, más flexible que otros lenguajes de programación de manera general, comienza Python a tomar un nuevo aire como uno de los lenguajes preferidos para la industria del desarrollo de software.

Historia

Python es un lenguaje de programación que se inició a principios de los años noventa, su creador es un holandés llamado Guido Van Rossum que se dedicaba a la investigación. Guido Van Rossum simplemente opto como muchos otros creadores de lenguajes de programación por utilizar uno diferente al que ya le habían sugerido usar, aunque en este caso el que él tenía en mente era uno propio. Entonces se decide crear un lenguaje por los detalles que habiá observado en un lenguaje de programación que se llama ABC.

Podemos decir que Python comenzó a ver la luz en el año 1991 pero en realidad solo lo primero que se publicó fue la versión 0.9, en ese momento aún faltaban cosas, pero ya se podía sentir la esencia de Python. Hasta el año 1994 es donde en realidad vemos la versión 1.0.

Podemos decir que Python cuenta con 2 versiones de forma general, la 3.x y la 2.x, definiendo X como un número cualquiera como por ejemplo: 2.6, 2.7, etc. Lo mismo pasaría con la versión 3.X. En este caso hay varias justificaciones para esto.

Variables

Las variables son "algo" que como su nombre indica puede variar, por ejemplo imaginamos una caja que cuando vamos a meter una pelota adquiere las dimensiones necesarias para guardarla, pero si fuéramos a necesitar almacenar un carro en esa caja, entonces cambiaría de tamaño para poder introducirlo sin problemas. Eso es una variable.

Las variables son una parte básica de la programación, ya que en la mayoría de las situaciones necesitariamos al menos una variable. Las variables pueden tomar diferentes formas o tamaños como se comentó en el párrafo anterior, la pregunta sería:

  • ¿Cuáles son las formas que puede tomar una variable?

La respuesta sería: Los tipos de datos que tenemos en cada uno de los lenguajes de programación son las "formas" que puede tomar una variable.

La unidad básica para medir una variable es el bit,aun que en realidad la mayoría lo que hacen es agrupar 8 bits para tomar como medida losbytes.

Es decir que por ejemplo el "tipo de dato A" puede tener de tamaño 4 bytes y el "tipo de dato B" puede tener el tamaño de 1 byte.

  • 1 byte = 8 bits
  • 4 bytes = 32 bits

Al asignarle nosotros el tipo de dato a una variable, le estamos diciendo a la computadora que asigne X cantidad de bits a nuestra variable y que nadie más puede tocar esa memoria hasta que finalicemos el programa, por eso no se recomienda declarar variables por declarar ya que estamos utilizando la memoria, un recurso muy valioso hoy en dia para el rendimiento de nuestros programas o aplicaciones.

Tipado

Tipado como la palabra lo indica hace referencia al tipo, por ahora esto seguramente no nos dice mucho, pero vamos a explicarlo hablando sobre los diferentes tipos que tenemos de tipado en la programación. Existe el tipado estático y el tipado dinámico, esto depende de cómo trabaja cada lenguaje de programacación, por ejemplo el lenguaje de programación Java trabaja con lo que se conoce cmo tipado estático, esto quiere decir que cuando una variable tiene un tipo, este se mantiene siempre y así es como trabaja.

Python por otro lado trabaja de manera distinta, utiliza el tipado dinámico. Esto quiere decir que una variable puede cambiar el tipo dependiendo del momento, además es el lenguaje el que se le asigna el tipo dependiendo de lo que "ve". Por ejemplo podríamos iniciar una variable "X" con el valor de 20 y esto haría que la variable "X" fuera del tipo entero, pero si después le asignamos otro valor, por ejemplo una cadena de texto como: "Esto es una cadena", entonces la variable X ahora tendría como tipo "String" que es una cadena

Números enteros

Los números incluyen varios tipos de datos, vamos a comenzar hablando de los enteros. En el lenguaje se conocen como "int".

Enteros

Lo que nos indica la palabra int cuando vemos un número es que tenemos un número entero, esto quiere decir que el número no está divido o fraccionado, por ejemplo un número entero sería:0, -10, 90, 999, -16

Lo que notamos a simple vista es que estos números no tienen decimales, así que si el tipo de dato que vamos a guardar necesita tener los decimales, podemos decir que este tipo de dato no es el que le corresponde. Vamos a crear un entero en Python cambiando la instrucción que teníamos para el "Hola mundo" por la siguiente: n =10

Float

Float es un tipo de dato que sigue dentro de los tipos de datos de numeros, este tipo de numeros son los que permiten tener numeros fraccionados o podemos decir que son numeros con decimales, por lo tanto son numeros que tiene mayor precisión, en algunos lenguajes de programación se les conoce como reales.

Float es una palabra inglesa y en español es común encontrar que se les mencione como números "flotantes" aunque el tipo directamente se llama "Float" y es muy raro que se utilice una traducción, un ejemplo de números flotantes serían estos: 1.89, 99.99, -10.90

Números_complejos

Ahora hablaremos de otro tipo de números que seguramente solo se escuchan en las universidades o institutos, los números complejos son números que están formados tanto por letras como números. En la teoría los números complejos son una extensión de los números reales y forman un cuerpo algebraicamente cerrado.

En Python vamos a encontrar los números complejos de manera fácil en el código cuando veamos la literal "j". Este es un ejemplo: 1 + 10j

Si queremos que el número complejo se encuentre en una variable, entonces haríamos lo siguiente: n = 1 + 10j
